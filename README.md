<h1 align="center" id="title">CLIMATE CHANGE &amp; QUALITY EDUCATION</h1>

<p id="description">Educational Website ‚Äì First Year Project is a static multi-page website designed to simulate an online learning platform. The project focuses on presenting educational content in a structured and user-friendly way using pure HTML and CSS.   ## Getting Started

These instructions will help you run the project locally.

Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- Optional: a simple HTTP server (like Live Server in VS Code) for correct routing

---

## Installation

1. Clone the repository:

   git clone https://github.com/adityaranjan05/Educational-Website-1st-Year-.git
2. Change into the project directory:

   cd Educational-Website-1st-Year-
3. Open `index.html` in your browser, or start a local server:

   - Using Python 3: `python -m http.server 8000`
   - Using Live Server (VS Code extension): Right-click `index.html` -> "Open with Live Server"
